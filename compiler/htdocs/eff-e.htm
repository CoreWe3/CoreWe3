The machine is Sun Fire V880 (4 Ultra SPARC III 1.2GHz, 8GB main memory, Solaris 9). OCamlOpt is version 3.08.3 and given the option "-unsafe -inline 100". GCC -m32 and GCC -m64 are version 4.0.0 20050319 and given the option "-O3". GCC -m32 -mflat is version 3.4.3 and given the option "-O3". MinCaml is given the option "-inline 100". It means to inline functions of less "size" than 100, not to repeat inlining for 100 times. Just for informal comparison, GCC has a similar option "--param max-inline-insns-auto=..." and its default is 100. See the distribution for details of each application.
